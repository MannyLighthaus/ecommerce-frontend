# E-commerce Frontend

A modern and responsive e-commerce frontend built with Vue 3, Composition API, Pinia for state management, Bootstrap for UI, and Vue Router for navigation. This is the frontend part of a full-stack e-commerce application.

## 📦 Technologies Used

- Vue 3 + Composition API
- Pinia (State Management)
- Vue Router
- Bootstrap 5
- BootstrapVue Next
- Netlify (Deployment)

## ⚙️ Getting Started

### 1. Clone the repository

````bash
git clone https://github.com/MannyLighthaus/ecommerce-.git
cd ecommerce-backend

## Project Setup

```sh
npm install
````

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
